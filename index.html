Rules

Rules are simple! There is a randomly generated sequence of 4 different colors at the beginning of the game. (Colors are blue, grey, yellow, red, orange, green, white and pink) Your goal is to try to guess the code! Simply create your sequence by selecting your four colors using the custom color selector, hit "Submit Guess," and you will be given clues as to how close your guess was to the actual code. Hints include:

    This will represent how many colors there in your code that appear in the actual code, but are not in the correct spot in the actual code

    This will represent how many colors there in your code that appear in the actual code that are also in the correct position in the actual code

    This will represent how many colors there in your code that do not appear in the actual code at all

You will have 10 guesses to crack the code. Guesses are reset after each guess. If you wish to restore a previous guess to think more about it, simply click the corresponding guess on the guess log, on the left side of the page. You cannot use duplicate colors, and you cannot guess the same sequence twice. Doing so will result in an error appearing on the screen. After you have either ran out of guesses or cracked the code, the game will end. IF you wish to play again, or simply restart the game you are currently in, simply click the "Start Over" button next to the "Submit Guess" Button. Good Luck!
